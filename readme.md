# 👗 Fashion Web Hub

*Fashion Web Hub* is a clean and simple e-commerce website template made for clothing businesses. It helps fashion brands or thrift sellers showcase their items and receive orders directly through WhatsApp.

---

## 📦 Features

- Mobile-friendly product cards
- Product name, image, and price
- Direct WhatsApp order buttons
- Easy to customize (HTML & CSS)
- Lightweight and fast
